Optional: The XML representing the CAML query you'd like to make of the documents in the document library. If not supplied, all documents will be retrieved.

Type: SharePointDocumentLibrary
A SharePointDocumentLibrary representing the library on the server, or null if the library could not be found, the user does not have sufficient permissions to access the library, or library's contents could otherwise not be retrieved.
